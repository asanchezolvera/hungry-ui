# Hungry UI

An Angular 19+ component library optimized for ecommerce applications.

## Quick Start

```bash
# Navigate to workspace
cd hungry-ui

# Install dependencies
npm install

# Start development
npm run storybook
```

## Documentation

See [hungry-ui/README.md](./hungry-ui/README.md) for detailed documentation.
